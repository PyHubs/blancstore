Orcim

Orcim is a simple IDLE alternitive for python beginers. Orcim is recommended for python beginers. Orcim is built by PyHubs on github.com. He made the text editor just for fun, and to see if he can make a good editor in tkinter.

Download Website Github

Orcim Features

    - Syntax highlighting for many languages
    - Two themes (monokai, dracula) (different themes coming soon*)
    - Easy to modify to your needs (copy the ORCIM.py and the LICENSE)

Upcoming Featues
