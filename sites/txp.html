TextEditor XP

TextEditor XP is a really simple text editor, or notepad alternitive. It supports python idlelib (idle) syntax highlighting and editing basic .txt files. This text editor will not recive any future ideas, you are free to fork it as long as you mention that it is forked from

TextEditor XP Features

    - Syntax highlighting using python idlelib
    - Can do all basic notepad tasks
    - Looks better than notepad
    - Easy to modify to your needs (copy the MAIn.py and the LICENSE) from the github
